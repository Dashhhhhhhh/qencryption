## Encrypt Your World

QEncryption is a powerful encryption tool designed to keep your data safe and secure. With its advanced algorithms and easy-to-use interface, you'll have full control over your digital information.

### Features

- **Strong Encryption**: Protect your data with top-level security.
- **Simple Interface**: Easy to use for everyone.
- **Cross-Platform**: Works on Windows, macOS, and Linux.
- **Open Source**: Transparent and community-driven.

### Installation

```sh
# Clone the repository
git clone https://github.com/yourusername/qencryption.git

# Navigate to the project directory
cd qencryption

# Install dependencies
npm install

# Run the application
npm start
```

### Usage

Encrypt a file:
```sh
qencryption encrypt myfile.txt
```

Decrypt a file:
```sh
qencryption decrypt myfile.txt.enc
```

### Using `interactive.py`

To use the `interactive.py` script for encryption and decryption, follow these steps:

1. Ensure you have Python installed on your system.
2. Navigate to the project directory.
3. Run the script with the following command:

```sh
python interactive.py
```

4. Follow the on-screen prompts to encrypt or decrypt your files interactively.
